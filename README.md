# 🛍️ Mini Katalog Uygulaması

Flutter ile geliştirilmiş, ürün listeleme ve sepet yönetimi yapabileceğiniz bir mobil alışveriş uygulaması.

## 📱 Özellikler

- Ürün listeleme (GridView)
- Ürün detay sayfası
- Sepete ekleme / çıkarma
- Toplam fiyat hesaplama
- Sayfa geçişleri (Navigator & Route Arguments)

## 🛠️ Kullanılan Teknolojiler

- Flutter 3.x
- Dart
- Material Design


## Çalıştırma Adımları

1. Repoyu klonlayın:
git clone https://github.com/Milarda/ilk_uygulamam.git

2. Proje dizinine girin:
cd ilk_uygulamam

3. Bağımlılıkları yükleyin:
flutter pub get

4. Uygulamayı çalıştırın:
flutter run
